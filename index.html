Conectar ao Firebase

Para funcionar online com múltiplos participantes, este quiz usa o Firebase Realtime Database (gratuito). Siga os passos abaixo uma única vez.

  1. Acesse console.firebase.google.com e crie um projeto gratuito.
  2. No menu lateral, clique em Realtime Database → Criar banco de dados. Escolha qualquer região e inicie em modo de teste (regras abertas por 30 dias).
  3. Clique na engrenagem (⚙) → Configurações do projeto → Seus apps → Web (</>). Registre o app e copie os dados de configuração.
  4. Cole os valores nos campos abaixo e clique em Salvar e entrar.
⚠️ Campo obrigatório — sem isso o banco de dados não funciona.
